Data Offerings

Quotail offers three different datasets that are delievered via
a REST API or flat file.

Raw Trade Data

A file with all raw options trades straight from the exchange. Also available as intraday delivery option.

  • Option Symbol
  • Linux Epoch Timestamp (millisecond precision)
  • Sequence Number
  • Quantity
  • Price
  • Bid & Ask
  • Aggressor Side
  • Spread Legs
  • Exchange

Top Seller

Enriched Trade Data

Option time and sales data that aggregates sweep orders, groups together spread legs, and classifies the aggressor side. This dataset includes everything from the raw trade data package PLUS what you see below.


  • Clustered Multi-Market Sweep Orders
  • Grouped Spread Legs
  • Implied Volatility
  • Greeks
  • Stock Price (at time of trade)
  • Volume
  • Open Interest

Options Chains Data

This data package includes all options contracts, such as index options and ETFs, and contains end-of-day bid and ask quotes, volume, open interest, and volatility.


  • Equity Options
  • Index Options
  • ETF Options
  • End-of-day Bid and Ask
  • Volume
  • Open Interest
  • Implied Volatility
